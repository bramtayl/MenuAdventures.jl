MenuAdventures.Parts

A sub-module that allows things to be part of other things.

julia> using MenuAdventures

julia> using MenuAdventures.Testing

julia> using MenuAdventures.Parts

        end;

        end;

        end;

julia> @noun struct StickyThing <: Noun
        end;

MenuAdventures.Parts.Attach — Type
Attach()

Attach something from your Inventory to something else Reachable.

An Action. By default, ever_possible(::Attach, ::Reachable, _) = true, that is, you can attach something sticky to anything.
